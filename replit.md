# Bot Project
## Overview
Facebook bot based on Mirai/FCA.

## Recent Changes
- Added 'المحرك' command for auto-reply in groups with on/off functionality.
- Added 'تكرار' command to rotate group name every 45 seconds with on/off functionality.

## Project Architecture
- `modules/commands/`: Command definitions.
- `main.js`: Main entry point (obfuscated).
- `index.js`: Starter script.
